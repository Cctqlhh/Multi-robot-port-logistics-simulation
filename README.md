# 思路梳理

## 1. 总体流程
    初始化 init
    循环：每一帧input()
        每一帧，每个机器人的操作：
            机器人获取全局地图 （全局地图：机器人、墙、水都为障碍物）
            __目标选取
            规划路径，输入目标
                判断目标是否改变，如果目标改变重新规划路线，目标不变可能调整路径
                    目标不变：判断下一位置是否存在机器人障碍，依据等待次数和机器人障碍决定移动操作
            移动操作转化为具体指令
                依据上一步规划的路径和等待条件将移动操作转化为给判题器的指令变量
            更新全局地图
